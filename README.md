# 每日鸡血 - 浏览器扩展

每天定时推送励志名言和故事，让你充满正能量！

## 功能特性

- **定时推送**: 每天定时收到励志名言或故事
- **自定义设置**: 可自由设置推送时间和频率
- **双重提醒**: 支持 Toast 通知和模态弹窗两种方式
- **自定义内容**: 可以添加、编辑、删除名言和故事
- **导入导出**: 支持数据备份和分享
- **在线抓取**: 可以从网页抓取励志内容
- **简洁美观**: 红黑配色，动画流畅

## 安装方法

### Chrome/Edge 浏览器

1. 打开 Chrome/Edge 浏览器，地址栏输入 `chrome://extensions`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目的 `daily-chicken` 文件夹
5. 安装完成，扩展图标会出现在工具栏

### Firefox 浏览器

1. 打开 Firefox，地址栏输入 `about:debugging`
2. 点击「此 Firefox」
3. 点击「临时载入附加组件」
4. 选择本项目中的 `manifest.json` 文件
5. 安装完成

## 使用说明

### 基本使用

1. 点击浏览器工具栏中的「每日鸡血」图标
2. 查看今日励志内容
3. 点击「换一批」随机更换内容
4. 点击「复制」将内容复制到剪贴板

### 设置方法

1. 点击扩展图标的「设置」按钮，或右键点击扩展图标选择「选项」
2. 在设置页面可以：
   - 开启/关闭每日提醒
   - 设置推送时间
   - 选择通知方式
   - 开启/关闭提示音

### 管理内容

1. 在设置页面的「内容管理」标签页
2. 可以添加、编辑、删除名言和故事
3. 名言支持设置作者
4. 故事支持设置标题和内容

### 导入导出

1. 导出：点击「导出数据」按钮，下载 JSON 文件
2. 导入：点击「导入数据」，选择备份的 JSON 文件
3. 恢复：点击「恢复默认」，还原初始数据

### 在线抓取

1. 在设置页面的「在线抓取」区域
2. 输入网页地址（支持豆瓣读书等页面）
3. 点击「抓取」按钮
4. 预览后可保存为名言或故事

## 技术栈

- **原生 JavaScript**: 无需构建工具
- **Chrome Extension API**: 使用 Manifest V3
- **CSS3**: 动画和过渡效果
- **Local Storage**: 本地数据存储

## 注意事项

1. 首次安装后需要手动启用扩展
2. 定时任务在浏览器运行时生效
3. 数据存储在浏览器本地，不会同步
4. 如遇问题，可尝试重新加载扩展

## 扩展权限说明

本扩展申请了以下权限：

- `alarms`: 用于定时发送通知
- `storage`: 用于存储设置和内容
- `notifications`: 用于显示系统通知
- `activeTab`: 用于获取当前标签页信息
- `host_permissions`: 用于抓取网页内容
